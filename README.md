# Homebrew TAPI

Homebrew tap for [TAPI](https://github.com/ksysoev/tapi) - Terminal API Explorer.

## Installation

```bash
brew tap ksysoev/tapi
brew install tapi
```

## Usage

```bash
# Explore an OpenAPI spec
tapi explore -f openapi.yaml

# Validate an OpenAPI spec
tapi validate -f openapi.yaml
```
